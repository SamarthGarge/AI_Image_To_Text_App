# ai_image_to_text_app

## Overview

`AI_Image_To_Text_App` is a Flutter application that converts images to text using AI. This project serves as a starting point for building more advanced image recognition and text extraction applications.

## Features

- Convert images to text using AI
- Cross-platform support (iOS, Android, Windows, Linux, macOS)
- Simple and intuitive user interface

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)
- [Android Studio](https://developer.android.com/studio) or [Visual Studio Code](https://code.visualstudio.com/)

### Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/SamarthGarge/AI_Image_To_Text_App.git
    ```
2. Navigate to the project directory:
    ```sh
    cd AI_Image_To_Text_App
    ```
3. Install dependencies:
    ```sh
    flutter pub get
    ```

### Running the App

To run the app on an emulator or physical device, use the following command:
```sh
flutter run
```
